# dundunkirk.github.io

Personal blog built with Hugo.

## About

This is my personal corner of the internet where I write about programming, open-source, and life. Topics include AI infrastructure, software engineering, and whatever else is on my mind.

## Stack

- Static site generator: [Hugo](https://gohugo.io/)
- Hosted on GitHub Pages
- Auto-deployed via GitHub Actions

## Local Development

```bash
hugo server -D
```

Then open http://localhost:1313.
