MineCraft 开服小白

发表于 2022-12-06 更新于 2022-12-07 分类于 🎮game

前期工作

JAVA

java下载，java向下兼容，因此直接下载最新版本，直接下载到默认文件夹（C盘），环境变量暂未更改

SPIGOT

SPIGOT https://getbukkit.org/download/spigot 按照需要的版本下载即可

准备好存放服务器数据的文件夹

新建txt文档，保存为strat.bat文件便于启动

java -Xms8G -Xmx8G -jar spigot-1.19.2.jar （文档内容）

启动start.bat

会生成两个重要文件，一个是eula.txt（协议，将false改为true），另一个是server.properties（配置文件）

再次启动start.bat

此时会生成地形，随机种子

输入stop可以保存停止

修改服务器参数 server.properties

MINECRAFT WIKI 中搜索server.properties

https://minecraft.fandom.com/zh/wiki/Minecraft_Wiki?variant=zh

镜像站 https://wiki.biligame.com/mc/Minecraft_Wiki

开启服务器

现在服务器已经开始啦

内网穿透

让不同WIFI下、不同地区的人都能加入游戏

注册

SAKURA FRP https://www.natfrp.com/

软件下载

在SAKURA FRP网站中，点击 服务-软件下载-启动器-Windows安装程序

新版本下下载的直接是exe文件，不需要自己解压等操作

进入软件

输入密钥

在SAKURA FRP网站中，点击 用户-用户信息，右下角一栏会出现密钥，将其输入至SAKURA FRP启动器中

创建隧道

新建隧道，选择进程名为java的进程

如果没有出现java进程，是因为本地服务器没有打开，即start.bat

打开隧道

在日志中可以查看服务器地址，现在就可以进行游戏啦
